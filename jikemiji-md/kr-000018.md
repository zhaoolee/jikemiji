---
title: npm临时使用淘宝源
---

最近两会,很多海外npm的源不容易连上,可以临时使用淘宝源

```
npm install
```
转换为
```
npm --registry https://registry.npm.taobao.org install
```


---


安装特定软件包也一样

```
npm install express
```
转换为
```
npm --registry https://registry.npm.taobao.org install express
```
